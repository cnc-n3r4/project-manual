## 2.


## Date

The NC's clock function continues counting time even when the power is turned off. Therefore, when the

power is turned on, the current date is displayed.


# I NOTICE I Since this date data is very important for the functions such as MacMan, alarm log, etc., do not

change it inadvertently.

Date setting procedure is given below.

(1)

Press function key [F1] (DATE).

The following is displayed on the console line and the system becomes ready for the input of date.

=DA

1994.9.30 FRIDAY 14:33:30

enter date (Y.M.D) !_

=-EX

=DA

1994. 9. 30 FRIDAY

enter date(Y.U.D)

MULTI

[EXTEND)

PIP

\

Press [F1] (DATE)

=DA

1994.9.30 FRIDAY

14:33:30

enter date (Y.M.D) 1 _

This ls only an example; current date,

day of week, and time a.re displayed.

(2)

Key in the year, month, and date with period put after each item following the prompt"!".

Example: 1994. 9.30

(3)

Press the WRITE key.

When the correct entry has been made, the display screen indicates date and a day of the week.

If the data entry is not correct, the display screen requests data entry again.

1.

[Supplement]

Entry of only right two digits is acceptable for year entry.

Pressing the WRITE key without entering the date ends the operation and nothing
